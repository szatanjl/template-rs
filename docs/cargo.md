cargo
=====


Commands
--------

- `cargo fetch`

  Fetch dependencies

- `cargo b[uild]`

  Build project binaries and library in debug mode

- `cargo b[uild] -r`

  Build project binaries and library in release mode

- `cargo b[uild] [-r] --bins`

  Build binaries

- `cargo b[uild] [-r] --lib`

  Build library

- `cargo package`

  Build cargo crate

- `cargo t[est]`

  Run tests

- `cargo r[un]`

  Build and run project

- `cargo clean`

  Remove built files
